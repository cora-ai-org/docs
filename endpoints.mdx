## Base Path
`/external-api/accounts`

## 1) Get One Account
`GET /external-api/accounts/:externalId`

Headers:

- `Authorization: Bearer <org_api_key>`

Curl:

```bash
curl -sS \
  -X GET "https://api.corafone.com/external-api/accounts/FILE_123" \
  -H "Authorization: Bearer cora_org_<keyId>.<secret>"
```

Success:

```json
{
  "success": true,
  "code": "ACCOUNT_FETCHED",
  "message": "Account fetched successfully",
  "account": {
    "id": "ACCOUNT_ID",
    "externalId": "FILE_123",
    "organizationId": "ORG_ID"
  }
}
```

## 2) Get Multiple Accounts
`GET /external-api/accounts?externalIds=FILE_1,FILE_2`

Headers:

- `Authorization: Bearer <org_api_key>`

Rules:

- `externalIds` is required
- Maximum 200 IDs per request

Curl:

```bash
curl -sS \
  -X GET "https://api.corafone.com/external-api/accounts?externalIds=FILE_1,FILE_2" \
  -H "Authorization: Bearer cora_org_<keyId>.<secret>"
```

Success:

```json
{
  "success": true,
  "code": "ACCOUNTS_FETCHED",
  "message": "Accounts fetched successfully",
  "accounts": [
    {
      "id": "ACCOUNT_ID",
      "externalId": "FILE_1",
      "organizationId": "ORG_ID"
    }
  ],
  "notFoundExternalIds": ["FILE_2"],
  "count": 1
}
```

## 3) Update One Account
`PATCH /external-api/accounts/:externalId`

Headers:

- `Authorization: Bearer <org_api_key>`
- `X-Cora-Timestamp: <unix_seconds_or_ms>`
- `X-Cora-Signature: <hmac_sha256_hex>`
- `Content-Type: application/json`

Request body (direct or `update` wrapper is supported):

```json
{
  "metadata": {
    "externalApiSyncState": "SYNCED"
  },
  "currentBalance": 450.25
}
```

Success:

```json
{
  "success": true,
  "code": "ACCOUNT_UPDATED",
  "message": "Account updated successfully",
  "account": {
    "id": "ACCOUNT_ID",
    "externalId": "FILE_123",
    "organizationId": "ORG_ID",
    "updatedAt": "2026-02-23T12:45:00.000Z"
  }
}
```

## Status Codes And Error Codes
### Typical statuses
- `200` success
- `400` invalid request body/query
- `401` missing/invalid auth or signature
- `404` account or route not found
- `405` method not allowed

### Common machine-readable error codes
- `EXTERNAL_IDS_REQUIRED`
- `EXTERNAL_IDS_LIMIT_EXCEEDED`
- `NO_VALID_UPDATABLE_FIELDS`
- `INVALID_JSON_BODY`
- `MISSING_AUTH_HEADER`
- `MISSING_AUTH_HEADERS`
- `INVALID_API_KEY`
- `REQUEST_TIMESTAMP_OUTSIDE_WINDOW`
- `INVALID_REQUEST_SIGNATURE`
- `ACCOUNT_NOT_FOUND`
- `METHOD_NOT_ALLOWED`
- `ROUTE_NOT_FOUND`
- `ACCOUNT_FETCH_FAILED`
- `ACCOUNTS_FETCH_FAILED`
- `ACCOUNT_UPDATE_FAILED`
