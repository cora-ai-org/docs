## Event
`payment.failed`

Sent when a payment attempt fails.

## Payload fields

- `title`
- `name`
- `fileNumber`
- `balance`
- `callId`
- `amount`
- `paymentDate`
- `note` (failure reason)
- `phoneNumber`
- `agentName`
- Optional: `amountPerInstallment`
- Optional: `numberOfInstallments`

## Example payload

```json
{
  "title": "Acme Corp Payment Failed",
  "name": "Jane Smith",
  "fileNumber": "12345",
  "balance": "1250.00",
  "callId": "call_abc123",
  "amount": "100.00",
  "paymentDate": "2026-01-30",
  "note": "Insufficient funds",
  "phoneNumber": "+15125550123",
  "agentName": "Collections Agent"
}
```
